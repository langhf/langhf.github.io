发表于 2018-07-19 | 分类于 Web |

最近在做服务器压测，想到可以通过 shell 脚本调用压测工具 ab 来进行多次测试。大致的想法是：

  1. 通过 shell 脚本循环调用 ab 压测工具得到多次压测日志，并分类保存；
  2. 通过 shell 脚本循环调用 awk 工具将每个日志中的 90% 时间提取出来。
